AGX Dynamics for Unreal is a plugin that integrates AGX Dynamics into Unreal Engine, enabling industry grade multi-purpose physics simulations.

# Links

Get it from [Fab](https://www.fab.com/listings/d62f8db4-b33b-440b-a23c-92f65d274fb3).

Instructions for how to build from source are in the [User Manual](https://us.download.algoryx.se/AGXUnreal/documentation/current/install.html#building-and-packaging-from-source).

The [Developer Guidelines](https://docs.google.com/document/d/e/2PACX-1vRGhEn_vJVekWtn43dLG-vAIllhXuiDAroPQSH1m8_OlrBUrSVOGJTIEf8fqj3ZfNRUBVE2MvCIvQ_p/pub) provides a starting point for developers who want to contribute to AGX Dynamics for Unreal.
It outlines the high-level architecture of the plugin, motivations for some of our decisions, describes the requirements we put on classes we add to the plugin, how we format and structure source code, how the code review process works, backwards compatibility, and things like that.
